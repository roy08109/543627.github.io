# Design System & Style Guide

## 1. Color Palette

### Primary Colors
- **Primary Blue**: `#0ea5e9` (Sky 500) - Used for primary actions, active states, and highlights.
- **Primary Glow**: `rgba(14, 165, 233, 0.5)` - Used for glow effects and shadows.
- **Primary Gradient**: `linear-gradient(135deg, #0ea5e9 0%, #3b82f6 100%)` - Used for buttons and featured icons.

### Secondary Colors
- **Accent Cyan**: `#22d3ee` (Cyan 400) - Used for price highlights and special accents.
- **Background Body**: `#020617` (Deep Navy/Black) - Main page background.
- **Background Surface**: `#0f172a` (Slate 900) - Component background.
- **Background Card**: `rgba(30, 41, 59, 0.4)` - Glassmorphism card background.

### Typography Colors
- **Text Main**: `#f8fafc` (Slate 50) - Primary headings and body text.
- **Text Secondary**: `#94a3b8` (Slate 400) - Subtitles and secondary information.
- **Text Muted**: `#64748b` (Slate 500) - Footer text and less important details.

## 2. Typography

### Font Families
- **Headings**: `Inter`, system-ui, sans-serif
- **Body**: `Inter`, system-ui, sans-serif (Fallback to system fonts)
- **Traditional Chinese**: `Noto Serif TC` (For specific cultural context)

### Font Sizes
- **Hero Title**: `56px` (Mobile: `36px`)
- **Section Title**: `2.2em`
- **Card Title**: `24px`
- **Body Text**: `16px`
- **Small Text**: `14px`

## 3. Spacing System (8px Grid)

- **XS**: `4px` (0.5x)
- **SM**: `8px` (1x)
- **MD**: `16px` (2x)
- **LG**: `24px` (3x)
- **XL**: `32px` (4x)
- **XXL**: `64px` (8x)

## 4. Layout

- **Container Width**: `1200px` (Centered)
- **Grid System**: Responsive Grid (Auto-fit, minmax 300px)
- **Header Height**: `72px` (Fixed)

## 5. Components

### Buttons
- **Border Radius**: `9999px` (Pill shape)
- **Padding**: `10px 24px`
- **Shadow**: `0 4px 12px rgba(14, 165, 233, 0.3)`

### Cards
- **Border Radius**: `24px`
- **Border**: `1px solid rgba(148, 163, 184, 0.1)`
- **Backdrop Filter**: `blur(10px)`
